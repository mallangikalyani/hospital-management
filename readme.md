# 🏥 Hospital Management System

A console-based Hospital Management System developed using Java and Object-Oriented Programming (OOP) concepts. The system allows users to manage patients, doctors, and appointments efficiently through a menu-driven interface.

## 🚀 Features

* Add Patient Records
* Add Doctor Records
* View Patient Details
* View Doctor Details
* Book Appointments
* View Appointment History
* Custom Exception Handling
* Menu-Driven Console Interface

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Collections Framework (ArrayList)
* Exception Handling
* Scanner Class

## 📂 Project Structure

```text
HospitalManagementSystem
│
├── Appointment.java
├── Doctor.java
├── Patient.java
├── HospitalService.java
├── HospitalException.java
└── Main.java
```

## 📚 OOP Concepts Implemented

### Classes and Objects

* Patient
* Doctor
* Appointment
* HospitalService

### Encapsulation

Private data members with public getter methods.

### Composition

Appointment class contains references to Patient and Doctor objects.

### Exception Handling

Custom exception:

```java
HospitalException
```

### Collections Framework

Uses ArrayList to store:

* Patients
* Doctors
* Appointments

## 📋 Functionalities

### Patient Management

* Add new patients
* View patient information

### Doctor Management

* Add doctors
* View doctor information

### Appointment Management

* Book appointments
* Validate patient and doctor availability
* View appointment records

## ▶️ How to Run

Compile all Java files:

```bash
javac *.java
```

Run the application:

```bash
java Main
```
